# IMC Prosperity 4 Visualizer

[![Build Status](https://github.com/mr-rambling/imc-prosperity-4-visualizer/workflows/Build/badge.svg)](https://github.com/mr-rambling/imc-prosperity-4-visualizer/actions/workflows/build.yml)

This repository contains the source code behind [mr-rambling.github.io/imc-prosperity-4-visualizer/](https://mr-rambling.github.io/imc-prosperity-4-visualizer/), a visualizer for [IMC Prosperity 4](https://prosperity.imc.com/) algorithms. 

It is based on jmerles visualizers for Prosperity [1](https://github.com/jmerle/imc-prosperity-visualizer), [2](https://github.com/jmerle/imc-prosperity-2-visualizer) and [3](https://github.com/jmerle/imc-prosperity-3-visualizer).
